Exemplos
Exemplo 1 – Raízes reais diferentes
f(x) = x² - 5x + 6
  • Identificando os coeficientes: a = 1, b = -5, c = 6
  • Calculando o discriminante (Δ):
    Δ = b² - 4ac
    Δ = (-5)² - 4(1)(6)
    Δ = 25 - 24
    Δ = 1
  • Fórmula de Bhaskara:
    x = (-b ± √Δ) / 2a
    x = (5 ± √1) / 2
    x = (5 ± 1) / 2
  • Raízes:
    x₁ = (5 + 1) / 2 = 6/2 = 3
    x₂ = (5 - 1) / 2 = 4/2 = 2
As raízes dessa equação são x = 2 e x = 3, ou seja, a parábola cruza o eixo x nesses dois pontos.
Exemplo 2 – Raízes reais iguais (Δ = 0)
f(x) = x² - 4x + 4
  • Identificando os coeficientes: a = 1, b = -4, c = 4
  • Calculando o discriminante (Δ):
    Δ = b² - 4ac
    Δ = (-4)² - 4(1)(4)
    Δ = 16 - 16
    Δ = 0
  • Quando Δ = 0, temos uma raiz real de multiplicidade 2:
    x = -b / 2a
    x = 4 / 2
    x = 2
A parábola toca o eixo x em apenas um ponto (x = 2), que é também o vértice da parábola. Quando Δ = 0, a parábola "beija" o eixo x.
Exemplo 3 – Foco no vértice
f(x) = -2x² + 8x - 5
  • Identificando os coeficientes: a = -2, b = 8, c = -5
  • Calculando as coordenadas do vértice:
    xv = -b/2a = -8/(-4) = 2
    yv = f(2) = -2(2)² + 8(2) - 5
    yv = -2(4) + 16 - 5
    yv = -8 + 16 - 5
    yv = 3
  • Como a < 0 (a = -2), a parábola tem concavidade para baixo
    Isso significa que o vértice é um ponto de máximo.
O vértice da parábola está em (2, 3), sendo este o ponto de valor máximo da função. Como a parábola possui concavidade voltada para baixo, ela atinge seu valor máximo neste ponto.
Exemplo 4 – Aplicação no cotidiano
h(t) = -5t² + 20t

Problema: Um objeto é lançado para cima com velocidade inicial. Sua altura em função do tempo é dada por h(t) = -5t² + 20t.

  • Identificando os coeficientes: a = -5, b = 20, c = 0
  • Para encontrar a altura máxima, calculamos o vértice:
    tv = -b/2a = -20/(-10) = 2 segundos
    hmax = h(2) = -5(2)² + 20(2)
    hmax = -5(4) + 40
    hmax = -20 + 40
    hmax = 20 metros
  • Para saber quando o objeto toca o chão, encontramos as raízes:
    h(t) = 0
    -5t² + 20t = 0
    t(-5t + 20) = 0
    t = 0 ou t = 4
O objeto atinge altura máxima de 20 metros após 2 segundos do lançamento. O tempo total até tocar o chão é de 4 segundos.
