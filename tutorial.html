Como calcular as parábolas?
  1. Entenda a equação

    A equação de uma parábola (função do 2º grau) tem a forma:

    f(x) = ax² + bx + c

    Onde:

    • a determina a concavidade (para cima se a > 0, para baixo se a < 0)
    • b e c influenciam o deslocamento da parábola
  2. Calcule o discriminante (Δ)

    Δ = b² - 4ac

    O valor de Δ te diz quantas raízes a parábola tem:

    • Δ > 0 → 2 raízes reais diferentes
    • Δ = 0 → 1 raiz real (vértice toca o eixo x)
    • Δ < 0 → nenhuma raiz real
  3. Aplique a fórmula de Bhaskara

    \( x = \frac{-b \pm \sqrt{\Delta}}{2a} \)

    Com isso, você encontra os pontos onde a parábola cruza o eixo x (se Δ ≥ 0).

  4. Encontre o vértice da parábola

    \( x_v = \frac{-b}{2a} \)

    yv = f(xv)

    O ponto (xv, yv) é o vértice, ou seja, o ponto máximo ou mínimo da parábola.

  5. Conclua a análise

    Com as raízes e o vértice, você já consegue:

    • Saber se a parábola abre para cima ou para baixo
    • Desenhar o gráfico corretamente
    • Resolver problemas de contexto (como altura de um objeto, área máxima, etc)
